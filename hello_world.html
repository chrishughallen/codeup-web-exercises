How to add/commit your work to GitHub

  • Create a new file or edit an existing file
  • Anytime you add new work, make a commit.
  • In IntelliJ - make a commit w/ Command+K
  • Add a 'commit message' that explains what changes you've made/what you're doing
  • Hover over the 'commit' button and select 'commit and push'
